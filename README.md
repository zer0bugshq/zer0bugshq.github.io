# BugHunter Tracker

A lightweight, privacy-focused dashboard built for bug bounty hunters, penetration testers, and security researchers.

BugHunter Tracker helps you organize vulnerability findings, track bug bounty earnings, maintain research worklogs, store proof-of-concepts, and manage writeups - all from a single interface.

## Features

### Vulnerability Management

* Track findings across platforms such as HackerOne, Bugcrowd, and private programs.
* Categorize vulnerabilities by severity (P1 - P5).
* Manage report status from discovery to resolution.
* Store affected assets, technical details, remediation notes, and references.

### Earnings Dashboard

* Monitor lifetime and monthly bounty earnings.
* Track progress toward financial goals.
* Analyze payouts by severity and program.

### Workspaces

* Create dedicated workspaces for individual targets.
* Maintain daily testing notes and research logs.
* Store payloads, requests, responses, and proof-of-concept data.
* Link workspace notes directly to findings.

### Notes & Writeups

* Draft vulnerability reports before submission.
* Organize methodologies, testing techniques, and private research notes.
* Maintain a personal knowledge base.

### Backup & Portability

* Export all data to JSON.
* Import backups on any device.
* No vendor lock-in.

## Privacy & Security

BugHunter Tracker is designed with privacy in mind.

* Runs entirely in the browser.
* No telemetry or analytics.
* No third-party tracking.
* AES-256 encryption for stored data.
* User-controlled encryption password.
* Cloud synchronization stores only encrypted data.

## Cloud Sync

Optional cloud synchronization is available through JSONBin.

* Sync encrypted data between devices.
* Data remains unreadable without your password.
* No plaintext records are stored remotely.

## Workflow

1. Create a workspace for a target.
2. Record testing notes and payloads.
3. Log verified findings.
4. Submit reports to the platform.
5. Track triage and resolution status.
6. Monitor earnings and payouts.
